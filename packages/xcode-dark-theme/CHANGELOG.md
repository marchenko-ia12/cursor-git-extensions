# Changelog

## 0.1.0

Initial release.

- Port of the `Xcode-Dark` color scheme by
  [Antelle](https://github.com/antelle/intellij-xcode-dark-theme) from
  the WebStorm / JetBrains plugin to VS Code and Cursor.
- UI colors (background `#323333`, selection `#1D7BED`, tab underline)
  and TextMate token scopes covering keywords, strings, classes,
  interfaces, functions, fields, globals, decorators, HTML / XML tags
  and more.
